## Descrição

Microsserviço de catálogo

## Back End

PHP: 7.3

## Banco

MySql: 5.7

## Rodar a aplicação

#### Crie os containers com Docker

```bash
$ docker-compose up -d
```

#### Accesse no browser

```
http://localhost:8000
```
#### Rodar testes

```
./vendor/bin/phpunit
```
